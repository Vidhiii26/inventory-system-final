# Inventory Management System

This is my full-stack assignment for the technical assessment. I built a simple inventory and order management system.

## Tech Stack Used
- Frontend: React + Vite
- Backend: FastAPI (Python)
- Database: PostgreSQL (Neon DB)
- Deployment: Vercel (Frontend) & Render (Backend)

## Live Links
- Frontend App: https://inventory-management-system-chi-six.vercel.app
- Backend API Docs: https://inventory-management-system-dd3t.onrender.com/docs

## How to run locally
1. Clone the repo
2. Run `docker-compose up --build`
3. Frontend will run on localhost:5173 and backend on localhost:8000.

(Note: I have used a cloud database so you don't need to configure local postgres).
